# Usability Testing Plan

## Test Objectives
1. Evaluate ease of loan application process
2. Identify confusion points in terminology
3. Measure user confidence in decision making

## Methodology
**Remote Moderated Testing** dengan 5 participants

## Participant Criteria
- Usia 22-35 tahun
- Pernah menggunakan aplikasi pinjaman
- Comfortable dengan mobile apps

## Tasks & Scenarios

### Task 1: Calculate Loan Scenarios
"Kamu ingin pinjam Rp 10 juta untuk biaya pendidikan. Coba hitung berapa angsuran per bulan untuk tenor 12 bulan."

**Success Criteria**:
- User bisa menemukan calculator
- Bisa adjust amount dan tenor
- Memahami hasil perhitungan

### Task 2: Submit Loan Application  
"Setelah memutuskan jumlah pinjaman, ajukan pengajuan dengan data berikut: [mock data]"

**Success Criteria**:
- Complete form dalam 5 menit
- Tidak lebih dari 2 errors
- Feel confident dengan keputusan

### Task 3: Check Application Status
"Cek status pengajuan yang sudah kamu submit tadi"

**Success Criteria**:
- Bisa menemukan status page
- Memahami informasi yang ditampilkan
- Tahu next steps

## Metrics to Track
- Time on task
- Error count
- Success rate
- Satisfaction score (1-5)
