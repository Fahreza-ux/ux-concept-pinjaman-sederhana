# User Journey Map

## Persona: Rina (Fresh Graduate)

### Stage 1: Awareness
**Touchpoint**: Lihat iklan di Instagram
**Feeling**: Interested but skeptical
**Opportunity**: Show simple app interface

### Stage 2: Download & Onboarding
**Touchpoint**: Download app dari Play Store
**Feeling**: Hopeful
**Pain Point**: Permissions yang banyak
**Improvement**: Minimal permissions, clear value proposition

### Stage 3: Loan Simulation
**Touchpoint**: Gunakan kalkulator pinjaman
**Feeling**: Empowered
**Opportunity**: Interactive slider untuk jumlah & tenor
**Success**: User paham komitmen financial

### Stage 4: Application Process
**Touchpoint**: Isi form pengajuan
**Feeling**: Anxious
**Pain Point**: Form terlalu panjang
**Improvement**: Progressive form dengan progress bar

### Stage 5: Waiting Period
**Touchpoint**: Cek status pengajuan
**Feeling**: Impatient
**Pain Point**: Status tidak jelas
**Improvement**: Real-time status dengan estimasi

### Stage 6: Fund Disbursement
**Touchpoint**: Terima dana
**Feeling**: Relieved
**Opportunity**: Celebration moment
